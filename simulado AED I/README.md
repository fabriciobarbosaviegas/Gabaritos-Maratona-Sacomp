# 📚 Simulado de AED I

Esta pasta reúne as soluções comentadas do **Simulado de Algoritmos e Estruturas de Dados I (AED I)** utilizado na preparação para a **I Maratona SACOMP de Programação**.

Os problemas foram selecionados para reforçar conceitos fundamentais de programação, lógica, estruturas de dados básicas e algoritmos clássicos.

## 🎯 Objetivos do Simulado

Os exercícios deste simulado abordam habilidades essenciais para iniciantes em programação competitiva:

- Manipulação de entrada e saída;
- Estruturas condicionais;
- Simulação;
- Pilhas;
- Algoritmos gulosos;
- Observações matemáticas;

---

## 🚀 Linguagens

As soluções foram desenvolvidas em:

- 🐍 Python
- 📘 C99 (quando aplicável)

## 📖 Sobre os READMEs

Cada problema possui um README contendo:

- resumo do enunciado;
- explicação da lógica;
- passo a passo da solução;
- exemplos de entrada e saída;
- observações sobre o algoritmo utilizado.

O objetivo é que o repositório sirva tanto como **gabarito** quanto como **material de estudo** para futuras maratonas e disciplinas de Algoritmos e Estruturas de Dados.
