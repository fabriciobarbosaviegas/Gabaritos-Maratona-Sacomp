# 🏆 Prova Oficial — I Maratona SACOMP de Programação

Esta pasta reúne os **gabaritos oficiais** da prova aplicada durante a **I Maratona SACOMP de Programação**, realizada em **26 de junho de 2026**.

Os problemas foram selecionados para avaliar conhecimentos de algoritmos, estruturas de dados, matemática computacional e técnicas clássicas de programação competitiva, contemplando desde questões introdutórias até problemas de nível intermediário.

## 🎯 Competências Avaliadas

A prova buscou avaliar competências comuns em competições de programação, incluindo:

* Observação matemática;
* Manipulação de strings;
* Estruturas de dados;
* Árvores e busca em profundidade (DFS);
* Operações bit a bit (Bitwise);
* Algoritmos gulosos;
* Teoria dos números;

---

## 🚀 Linguagens

As soluções oficiais foram implementadas em:

* 🐍 Python
* 📘 C99

Alguns problemas possuem implementações em ambas as linguagens, permitindo comparar diferentes abordagens e estilos de implementação.

## 📖 Sobre os READMEs

Cada problema possui um README contendo:

* resumo do enunciado;
* explicação detalhada da lógica utilizada;
* ilustrações quando aplicável;
* análise de complexidade;
* exemplos de entrada e saída;
* observações sobre possíveis otimizações.

O objetivo deste material é servir como **gabarito oficial da prova**, além de funcionar como material de consulta para estudantes interessados em programação competitiva e preparação para futuras edições da Maratona SACOMP.
