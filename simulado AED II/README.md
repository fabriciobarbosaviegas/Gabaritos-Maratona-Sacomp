# 📘 Simulado AED II

Esta pasta reúne os **gabaritos do Simulado de Algoritmos e Estruturas de Dados II (AED II)**, elaborado como atividade avaliativa e preparação para competições de programação, como a **Maratona SBC** e **Maratona SACOMP**.

Cada problema é acompanhado por uma solução em **Python** ou **C99**, além de um **README** contendo uma explicação detalhada do algoritmo utilizado.

## 📖 Estrutura de cada problema

Cada diretório possui a seguinte organização:

```text
<id do problema>/
├── <id>.py ou <id>.c
└── README.md
```

Os READMEs incluem:

* 📜 Resumo do enunciado;
* 🧠 Explicação da estratégia utilizada;
* 🎯 Exemplos ilustrativos;
* 📊 Análise de complexidade;
* ▶️ Forma de compilação e execução;
* 💡 Exemplos de entrada e saída.

---

## ▶️ Como executar

### Python

```bash
python3 arquivo.py
```

### C99

```bash
gcc -std=c99 arquivo.c -o programa
./programa
```
